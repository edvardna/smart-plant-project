# 🌱 Smart Plant Monitoring System

An embedded system designed to monitor plant conditions and support automated plant care using sensors and real-time data.

---

## 🚀 Overview
The Smart Plant System is a sensor-based project that monitors environmental conditions such as soil moisture (and optionally temperature/humidity) to help maintain optimal plant health.

The system collects real-time data and can be extended to trigger automatic actions, such as watering, making it a foundation for smart home and IoT-based agriculture solutions.

---

## 🧠 Features
- Real-time soil moisture monitoring  
- Environmental sensing (temperature / humidity)  
- Threshold-based decision logic  
- Automatic watering system  
- Simple and efficient embedded design  

---

## 🛠 Technologies Used
- **Microcontroller (Arduino / ESP32)**  
- **C / C++** for embedded programming  
- **Soil moisture sensor**  
- **Temperature / humidity sensor**  
- **Relay / pump module**  
- Basic electronics and circuit design  

---

## ⚙️ System Architecture
- **Sensors**
  - Measure soil moisture and environmental conditions  

- **Microcontroller**
  - Reads sensor data  
  - Processes values  
  - Applies threshold logic  

- **Output / Actuation**
  - Activates watering system when needed  

---

## 🎯 Purpose
The goal of this project was to:
- Learn how to interface **analog and digital sensors**
- Build a **data-driven embedded system**
- Explore **automation using threshold-based control**
- Understand how embedded systems can be applied to **real-world problems**
